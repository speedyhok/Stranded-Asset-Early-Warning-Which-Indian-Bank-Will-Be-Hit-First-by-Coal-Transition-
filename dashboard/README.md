## Overview
This is the interactive React + Vite web dashboard for the **Climate Credit Deterioration & Bank Contagion Predictor**. It imports the simulation data and models generated by the backend engine to provide an intuitive, high-fidelity visualization of climate transition risks.

### Visualized Features
1. **Corporate Risk Analytics:** Interactive charts detailing Carbon Beta, Cumulative Abnormal Returns (CAR) from Event Studies, and NLP sentiment metrics, compounding into the Transition Vulnerability Score (TVS).
2. **Survival Functions:** Visual default probability (PD) curves showing how conditional default curves shift over time (12M, 18M, 24M, 36M horizons).
3. **NGFS Scenario Modeling:** Side-by-side comparison of the banking sector's capital adequacy trajectory under Net Zero 2050, Delayed Transition, and Business-As-Usual (BAU) pathways.
4. **Interbank Contagion Propagation:** Node-link network representations of bank distress propagation (GNN-modeled CET1 ratio capital degradation and contagion waves).

## Installation & Getting Started

### Prerequisites
- Node.js (version 16 or higher)
- npm or yarn

### Dashboard Pipeline Ingestion
Before running the dashboard, ensure you have generated the baseline datasets from the python pipeline:
```bash
# From the project root directory
python prepare_web_dashboard_data.py
```
This script exports the compiled JSON datasets into the frontend src directory at:
`dashboard/src/data/modelData.js`

### Run locally
1. Navigate to the dashboard directory:
   ```bash
   cd dashboard
   ```
2. Install npm dependencies:
   ```bash
   npm install
   ```
3. Start the local Vite development server:
   ```bash
   npm run dev
   ```
4. Open the browser and visit the local dev server link (usually `http://localhost:5173/`).

## Project Structure (Frontend)

- **`src/App.jsx`:** The core dashboard component housing layout, state management, chart configuration (using Recharts/ChartJS/D3), and network visualizers.
- **`src/App.css`:** Core stylesheets containing layouts, color tokens, and responsive web design media queries.
- **`src/data/modelData.js`:** The auto-generated data module storing the backend's model coefficients, survival statistics, bank network parameters, and scenario histories.
